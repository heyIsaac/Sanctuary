1. ESTRUTURA DE PASTAS (ASSETS)
-------------------------------
Pastas do próprio jogo (conteúdo KillZombies):
  Assets/Scenes     → Cenas (SampleScene = principal)
  Assets/Scripts   → Lógica do jogo (ArmaJogador, ZumbiVida, Porta, etc.)
  Assets/Prefabs   → Prefabs do projeto (zumbi, animator)
  Assets/Sounds    → Áudio (trilha, efeitos)

Pastas de configuração e ferramentas:
  Assets/Settings  → URP e perfis de renderização
  Assets/Editor    → Scripts de editor (ex.: patch Linux para Meta XR)

Assets de terceiros (mantidos em pastas próprias para fácil identificação):
  Assets/StarterAssets           → Template Unity (FPC, ambiente, input)
  Assets/Polytope Studio         → Ambiente lowpoly (cenário, vila)
  Assets/Supercyan Character Pack Zombie Sample  → Zumbis
  Assets/LP_SciFiWeapons_Lite   → Armas
  Assets/TutorialInfo            → Readme do template (opcional manter)
  Assets/XR                      → Configurações XR (Oculus)
