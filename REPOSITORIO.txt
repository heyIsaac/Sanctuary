IMPORTANTE PARA A ENTREGA
-------------------------
O repositório deve conter apenas as pastas necessárias:
  - Assets
  - ProjectSettings
  - Packages

Não é necessário (e não deve) incluir:
  - Library
  - Logs
  - Temp
  - obj
  - Build/Builds
  - UserSettings

O arquivo .gitignore do projeto já está configurado para ignorar essas pastas.
Ao fazer clone ou download do repositório, abra o projeto no Unity Hub para
que a pasta Library seja gerada automaticamente na primeira abertura.
